# react-crypto-dashboard
> Built a dashboard to view realtime price actions and charts of top Cryptocurrencies using React

##Technologies used
1. React
2. Styled Componenet
3. HighCharts
4. lodash
5. moment
6. fuzzy search

API: https://www.cryptocompare.com/
